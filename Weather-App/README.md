# Seasonal Weather Desk

A lightweight browser app that:

- shows live temperature in `deg C` and `deg F`
- lets the user choose any country from a full selection list, with India selected by default
- fetches live weather for the entered city
- explains the current situation in plain language
- shows seasonal timing
- lists practical precautions for each season

## How to run

1. Open `index.html` in a browser, or serve the folder with a simple static server.
2. Enter a city name.
3. Keep or change the country selection.
4. Click `Get weather`.

## Data source

The app uses the public Open-Meteo Geocoding API and Open-Meteo Weather Forecast API for live weather data, so no API key is required.
