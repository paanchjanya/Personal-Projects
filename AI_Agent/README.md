# AI Agent Web Interface

This is a web-based version of your AI Agent.

## How to Run

1.  Open a terminal in this directory (`e:\AI_Agents`).
2.  Run the server:
    ```bash
    node server.js
    ```
3.  Open your browser and navigate to:
    [http://localhost:3000](http://localhost:3000)

## Features

*   **Modern UI**: Glassmorphism design with animated background.
*   **Interactive**: Chat-like interface to define words.
*   **Real-time**: Connects to Gemini API directly from the browser.

## Files

*   `server.js`: Simple web server to serve the files.
*   `public/index.html`: The main page.
*   `public/style.css`: Styling and animations.
*   `public/script.js`: Logic to communicate with Gemini.
